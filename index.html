คำแนะนำการนำแรงไปออกแบบจุดเชื่อมต่อ (Connection Design)

ในทางวิบัติวิทยา โครงสร้างบันไดมักจะวิบัติที่จุดเชื่อมต่อก่อนตัวแม่บันไดเสมอ การนำค่าจากโปรแกรมไปออกแบบจุดต่อมีแนวทางดังนี้

1. จุดรองรับที่ฐานบันได (Base Support / Base Plate)

คือจุดที่โคนบันไดเหล็กยึดติดกับพื้นตอม่อคอนกรีต หรือคานเหล็กชั้นล่าง

  • $R_y$ (แรงปฏิกิริยาแนวดิ่ง): ใช้คำนวณหากำลังรับแรงเฉือนของรอยเชื่อม, แรงเฉือนใน Anchor Bolt และ Bearing (การกดทับ) บน Base Plate
  • $H_x$ (แรงถีบแนวราบ): ถ้าจุดรองรับเป็น Fixed หรือ Pinned จะต้องนำแรงนี้ไปคำนวณแรงเฉือน (Shear) หรือแรงดึง (Tension) ในน็อตฝังด้วย (แรงนี้มักจะทำให้จุดต่อวิกฤต)
  • $M_z$ (โมเมนต์ที่ฐาน): ถ้าจุดรองรับเป็น Fixed ต้องนำค่าไปหาความหนาแผ่นเหล็ก Base Plate และคำนวณแรงดึง-อัดในกลุ่มน็อตฝัง (Anchor Bolt)
สิ่งที่จะต้องคำนวณต่อ: ขนาดและความหนาของ Base Plate, ขนาด/ระยะฝังของ Anchor Bolt, ขนาดรอยเชื่อม (Fillet Weld)

2. จุดรองรับด้านบน (Top Connection)

คือจุดที่ชานพักบันไดยึดเข้ากับคานหลักของตัวอาคาร (คานเหล็กหรือคานคอนกรีต)

  • ใช้หลักการเดียวกับฐานบันไดคือดึงค่าปฏิกิริยา $R_y, H_x, M_z$ ไปใช้งาน
⚠️ ข้อควรระวังสูงสุด: คานของตัวอาคารที่มารับบันได จะต้องถูกออกแบบให้รับแรง $H_x$ (แรงถีบ) จากบันไดด้วย ซึ่งจะทำให้คานอาคารเกิด "การดัดในแกนอ่อน (Weak-axis Bending)" หรือ "แรงบิด (Torsion)" หากไม่ได้เตรียมคานอาคารไว้รับแรงถีบ แนะนำให้ออกแบบจุดรองรับบันไดฝั่งใดฝั่งหนึ่งเป็นจุดเลื่อน (Roller) เพื่อให้ $H_x = 0$

3. รอยต่อจุดหักมุม (Kink Point / Splice Connection)

จุดที่ช่างจะต้องทำการ "ตัดบาก (Miter Cut)" และเชื่อมต่อเหล็กกล่องให้หักมุมตามรูปทรงบันได

  • $M_{kink}$ (โมเมนต์ที่จุดหักมุม): ใช้คำนวณหาแรงดึง/แรงอัดในปีกคาน ($P_f$) เพื่อประเมินพื้นที่หน้าตัดของรอยเชื่อม
  • $F_r$ (Radial Force): ใช้สำหรับออกแบบความหนาของแผ่น Stiffener Plate ดามด้านใน เพื่อป้องกันไม่ให้มุมคานเหล็กกล่องฉีกขาดเสียรูป
  • $V_u$ และ $P_u$ (Local Forces): นำค่าแรงเฉือนและแรงตามแนวแกนบริเวณรอยต่อ (ดูได้จากแท็บไดอะแกรม) มารวมกับการเช็ครอยเชื่อม
สิ่งที่จะต้องคำนวณต่อ: ขนาดรอยเชื่อมต่อชน (แนะนำให้เป็น CJP - Complete Joint Penetration) และการเชื่อมยึดแผ่น Stiffener

สรุปการดึงค่าไปใช้ (Cheat Sheet)

ชิ้นส่วนที่ต้องการออกแบบต่อ ค่าที่ต้องดึงจากโปรแกรมไปใช้ (ดูในรายการคำนวณ)
น็อตฝัง (Anchor Bolts) / รอยเชื่อมโคนบันได แรงปฏิกิริยารวม: $R_y, H_x, M_z$
ความหนาแผ่นเพลท (Base Plate) แรงกดและโมเมนต์: $R_y, M_z$
คานอาคารที่มารับบันได (Supporting Beam) Point Load จากบันได: $R_y, H_x$
รอยต่อเชื่อมบากมุม (Kink Splice Weld) แรงภายในชิ้นส่วน: $M_{kink}, V_{local}, P_{local}$
แผ่นดามปีกคาน (Stiffener Plate) แรงลัพธ์แนวฉาก: $F_r$ (โปรแกรมคำนวณความหนาและพิกัดให้แล้ว)
